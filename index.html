O que é a HTML?👀

voltar

HTML e a abreviatura de HyperText Markup Language(Linguagem de marcação de Hypertexto), criada por Tim Berners-Lee, o mesmo que criou o WWW(World Wide Web).

A HTML é uma tecnologia desenvolvida para facilitar a visualização, navegação e leitura de informações em hypertexto.

O que é HTML semântica?📝

voltar

HTML semântica eo código escrito com tags que possuem significado, ou seja são tags que focam no significado em que cada parte do site executa, deixando a parte das formas(Cores, tamanhos etc) para as CSS.

Algumas tags úteis na HTML.✨

voltar

Aqui irei listar algumas tags que aprendi na DIO no curso Formação HTML Web Developer e que são muito utilizadas por desenvolvedores de sites.

<HTML> </HTML>
A principal tag dessa tecnologia, sem ela praticamente não existe um site.
<H1> </H1> até <H6> </H6>
as "H" ou (Heading Tags) são tags especificas para os cabeçalhos de textos.
<STRONG> </STRONG>
A tag Strong serve para deixar a(s) palavra(s) em negrito.
<EM> </EM>
A tag em serve para deixar a(s) palavra(s) em itálico.
<P> </P>
Serve para criar um paragrafo.
<Q> </Q>
Serve para criar uma citação.
<BLOCKQUOTE> </BLOCKQUOTE>
Serve para criar um bloco de citação.
<MARK> </MARK>
Essa tag tem como funcionalidade a marcação de texto.
<TITLE> </TITLE>
Serve para a criação do título do site.
<ABBR> </ABBR>
Em HTML é usada para representar uma abreviação ou acrônimo.
<UL> </UL>
Tag muito útil, pois com essa tag você consegue criar uma lista não ordenada.
<OL> </OL>
Outra tag muito útil, pois com essa tag você consegue criar uma lista ordenada.
